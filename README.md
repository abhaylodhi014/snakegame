# Snake Game

A classic **Snake Game** built with **JavaScript**, **HTML**, and **CSS**. The game records the player's score and high score. As the player eats food, the snake's length increases. The game ends if the snake touches the boundary or itself.

## Features

- **Score Tracking**: The game records your current score and high score.
- **Snake Growth**: Each time the snake eats food, its length increases.
- **Game Over Conditions**: The game ends if the snake collides with the boundary or itself.
- **Responsive Design**: Playable on both desktop and mobile devices.
- **Simple and Fun**: A classic game with simple mechanics that is easy to learn but hard to master.

## How to Play

1. **Start the Game**: Press the "Start" button to begin playing.
2. **Move the Snake**: Use the arrow keys (Up, Down, Left, Right) to control the snake.
3. **Eat Food**: The snake grows longer each time it eats food.
4. **Avoid Collisions**: Don't let the snake touch the boundary or its own body.
5. **Game Over**: The game ends if the snake collides with the boundary or itself. Your score will be displayed, and you can try again to beat your high score.

## Technologies Used

- **JavaScript**: For the game logic and interactivity.
- **HTML/CSS**: For the structure and styling of the game.
